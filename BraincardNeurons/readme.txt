Copy the BrainCardNeurons folder to MyDocuments\\Arduino\\libraries

Plug the BrainCard on an Arduino compatible board. Depending on your hardware, you may need spacers or an intermediary proto shied to ensure proper assembly.

Launch the Arduino IDE
Select the proper board under Tools/Board Manager menu
Select the SimpleScript example from the File/Examples/BrainCard menu

Refer to the comments inserted in the source code for a technical description of the scripts supplied in the examples folder.

Refer to http://general-vision.com/documentation/TM_NeuroMem_Technology_Reference_Guide.pdf for description of the neurons.

For more information and access to the latest software, go to https://www.general-vision.com/products/subscribe/
